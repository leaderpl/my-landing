## 📊 데이터 구조

1. **inquiries** - 강의 문의
2. **reviews** - 수강생 리뷰
3. **references** - 강의 레퍼런스

### API Endpoints
```
GET  /tables/references?limit=100
GET  /tables/reviews?limit=100
POST /tables/inquiries
POST /tables/reviews
```
